# Crypto-Trading
This is a crypto trading app where users can create account, make watchlist, buy/sell assets and many more. (Work in Progress...)
# Project Structure: 

1.Client
  - NextJs
  - React & Redux
  - Hooks & ContextAPI
  - Typescript
  - SASS & Tailwind
 
2.Server (Microservice architecture)
  - NodeJs(Express)
  - MongoDB & Mongoose
  - Typescript
 
3.Containerize & Orchestration & Automation
   - Docker
   - Kubernetes
   - Skaffold
